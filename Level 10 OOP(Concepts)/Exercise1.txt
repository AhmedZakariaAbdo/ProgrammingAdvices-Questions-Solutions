/*
Exercise 1 – Person Class
Overview

This program demonstrates the concept of Encapsulation in Object-Oriented Programming (OOP).

A class named clsPerson is created to represent a person. The class stores personal information such as:

ID
First Name
Last Name
Full Name
Email
Phone Number

It also provides methods to:

Access and modify data safely using Getters and Setters.
Send an Email.
Send an SMS.
Print all person's information.
OOP Concepts Used
1. Class

The program defines a class called clsPerson.

class clsPerson
{
    ...
};

The class acts as a blueprint for creating person objects.

2. Private Members

The data members are declared as private.

private:
short _ID;
string _First_Name;
string _Last_Name;
string _Full_Name;
string _Email;
string _Phone;

This prevents direct access from outside the class and protects the data.

Example:

Person1._First_Name = "Ahmed"; // Not Allowed
3. Encapsulation

Encapsulation means hiding data and allowing access through public methods.

Example:

void SetFirstName(string FirstName)
{
    _First_Name = FirstName;
}

string GetFirstName()
{
    return _First_Name;
}

The user cannot access _First_Name directly, but can use:

Person1.SetFirstName("Ahmed");
cout << Person1.GetFirstName();
4. Constructor

The constructor initializes the object when it is created.

clsPerson(int Id, string FirstName,
          string LastName,
          string FullName,
          string Email,
          string Phone)
{
    _ID = Id;
    _First_Name = FirstName;
    _Last_Name = LastName;
    _Full_Name = FullName;
    _Email = Email;
    _Phone = Phone;
}

When creating an object:

clsPerson Person1(
    10,
    "Ahmed",
    "Zakaria",
    "Ahmed Zakaria",
    "Ahmed@gmail.com",
    "8789266");

All person information is initialized immediately.

5. Read-Only Property

The comment:

short _ID; // read only

indicates that the ID should not be changed after object creation.

The class provides only a getter:

int GetID()
{
    return _ID;
}

and does not provide a setter.

This makes the ID effectively read-only.

6. Methods (Behaviors)

The class contains methods that represent actions a person can perform.

Send Email
void SendEmail(string subj, string Cont)
{
    cout << "\nThe Following Message Sent Successfully to Email: "
         << _Email << endl;

    cout << "Subject: " << subj << endl;
    cout << "Body: " << Cont << endl;
}

Example:

Person1.SendEmail("Hi", "How Are You?");

Output:

The Following Message Sent Successfully to Email:
Ahmed@gmail.com

Subject: Hi
Body: How Are You?
Send SMS
void SendSMS(string cont)
{
    cout << "\nThe Following SMS Sent Successfully To Phone: "
         << _Phone << endl;

    cout << cont << endl;
}

Example:

Person1.SendSMS("How Are You?");

Output:

The Following SMS Sent Successfully To Phone:
8789266

How Are You?
7. Print Method

The Print() method displays all person information.

void Print()
{
    ...
}

Example:

Person1.Print();

Output:

Info:
____________________
ID           : 10
FirstName    : Ahmed
LastName     : Zakaria
FullName     : Ahmed Zakaria
Email        : Ahmed@gmail.com
Phone        : 8789266
____________________
Real-World Company Example

Imagine a CRM (Customer Relationship Management) system used by companies such as Salesforce or HubSpot.

Each customer can be represented as an object:

clsPerson Customer1;

The system stores customer information:

Customer ID
Name
Email
Phone

The company can then:

View customer information.
Send promotional emails.
Send SMS notifications.
Update contact details.

This is exactly the same idea implemented in this exercise, but on a much smaller scale.

Summary

This exercise demonstrates:

Class Creation
Objects
Constructors
Private Members
Encapsulation
Getters and Setters
Read-Only Data
Methods (Functions)
Object Initialization
Printing Object Information

It is a practical example of modeling a real-world Person/Customer Entity using Object-Oriented Programming principles.



*/



#include <iostream>
using namespace std;

class clsPerson{

    private:
    short _ID;// read only
    string _First_Name;
    string _Last_Name;
    string _Full_Name;
    string _Email;
    string _Phone;


    public:
    
    clsPerson(int Id, string FirstName,string LastName,string FullName,string Email,string Phone){

        _ID = Id;
        _First_Name = FirstName;
        _Last_Name = LastName;
        _Full_Name = FullName;
        _Email = Email;
        _Phone = Phone;
    }

    void SetFirstName(string FirstName){

        _First_Name = FirstName;
    }

    string GetFirstName(){

        return _First_Name;
    }
    
    void SetLastName(string LastName){

        _Last_Name = LastName;
    }

    string GetLastName(){

        return _Last_Name;
    }
    
    void SetFullName(string FullName){

        _Full_Name = FullName;
    }
    
    string GetFullName(){

        return _Full_Name;
    }

    void SetEmail(string Email){
        
        _Email = Email;
    }
    
    string GetEmail(){

        return _Email;
    }
    
    void SetPhone(string phone){

        _Phone = phone;
    }
    
    string GetPhone(){

        return _Phone;
    }

    void SendEmail(string subj,string Cont){

        cout<<"\nThe Following Message Sent Successfully to Email: "<<_Email<<endl;
        cout<<"Subject: "<<subj<<endl;
        cout<<"Body: "<< Cont<<endl;
    }
    
    void SendSMS(string cont){

        cout<<"\nThe Following SMS Sent Successfully To Phone: "<<_Phone<<endl;
        cout<<cont<<endl;
    }

    int GetID(){

        return _ID;
    }

    void Print(){

        cout<<"Info:\n";
        cout<<"____________________"<<endl;
        cout<<"ID           : "<<_ID<<endl;
        cout<<"FirstName    : "<<_First_Name<<endl;
        cout<<"LastName     : "<<_Last_Name<<endl;
        cout<<"FullName     : "<<_Full_Name<<endl;
        cout<<"Email        : "<<_Email<<endl;
        cout<<"Phone        : "<<_Phone<<endl;
        cout<<"____________________"<<endl;
    }
};


int main() {

clsPerson Person1(10,"Ahmed","Zakaria","Ahmed Zakaria","Ahmed@gmail.com","8789266");

Person1.Print();

Person1.SendEmail("Hi","How Are You?");
Person1.SendSMS("How Are You ?");



return 0;

}
